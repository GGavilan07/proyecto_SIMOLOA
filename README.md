# proyecto_SIMOLOA
Este fue un proyecto realizado para el curso Gestión de Base de Datos.
El objetivo del trabajo es diseñar e implementar una solución de base de datos para el
sistema SIMOLOA, aplicando los contenidos desarrollados durante el módulo de Gestión
de Bases de Datos.
En este proyecto encontraremos consultas SQL, implementación de triggers, datos simulados, etc.
